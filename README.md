### 主题作者说明

Yilia 是[litten](http://litten.github.io/) 为 [hexo](https://github.com/tommy351/hexo) 2.4+制作的主题。
崇尚简约优雅，以及极致的性能。         

关于主题：

1. 本主题原作者是litten，不是nooldey。了解主题原作内容请查看[yilia by litten](https://github.com/litten/hexo-theme-yilia)
